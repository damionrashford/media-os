# Contributor tooling — not distributed

The skills in this folder are **development-only**. They help maintain `media-os`; they are NOT part of the plugin users install.

The plugin's user-facing surface lives at [`/skills/`](../../skills/) at the repo root — 96 production skills + 13 workflow skills, all loaded under the `/media-os:` namespace after `/plugin install media-os@media-os`.

If you landed here from GitHub's directory browser looking for what the plugin actually ships, go to [`skills/`](../../skills/).
