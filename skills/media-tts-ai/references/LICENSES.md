## Voice-clone ethics

These models can convincingly clone any voice from a short clip. The skill imposes **no technical restriction**, but the user is responsible for:

1. Obtaining explicit consent from any person whose voice is being cloned.
2. Complying with jurisdiction-specific laws (e.g. EU AI Act, California SB 942, Tennessee ELVIS Act).
3. Disclosing AI-generated voice in commercial / political content as required by local law.
4. Not using cloned voices for impersonation, fraud, or defamation.

The `clone` subcommand logs the reference-clip path in the output manifest as a minimal provenance trail.

## Synthetic-output copyright

Output audio generated by these models is under the same permissive license as the model — you own usage rights — **BUT** if you cloned a specific speaker, the speaker's right-of-publicity / voice likeness may still apply regardless of the model license. Written consent is the safest route for any commercial voice clone.
